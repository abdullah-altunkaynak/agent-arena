# SEO Yapılacaklar Listesi (Öncelik Sırasına Göre)

## P0 - Olmazsa Olmaz

- [x] Canonical URL stratejisini belirle ve indexlenebilir tüm sayfalarda benzersiz canonical etiketi üret.
- [x] Tarama kuralları ve sitemap adresini içeren robots.txt ekle.
- [x] Yayınlanmış blog yazıları ve kritik statik sayfaları içeren dinamik sitemap.xml ekle.
- [x] Tüm blog detay sayfalarında benzersiz title ve meta description zorunluluğu uygula.
- [x] Blog detay sayfalarına Open Graph ve Twitter Card etiketleri ekle, görsel fallback kullan.
- [x] Draft, admin, preview ve sadece dahili kullanıma açık rotalara noindex ve robots engeli uygula.

## P1 - Yüksek Etki

- [ ] Blog detay sayfalarına BlogPosting/Article JSON-LD ekle.
- [ ] Blog liste ve blog detay sayfalarına BreadcrumbList JSON-LD ekle.
- [ ] TR/EN sayfalar için hreflang alternatiflerini ekle.
- [ ] Slug değişiklikleri için URL geçiş politikasını ve 301 yönlendirmelerini tanımla.
- [ ] Soft-404 davranışını düzelt ve eksik içeriklerde doğru 404 yanıtı verildiğini doğrula.

## P2 - Optimizasyon

- [ ] Dahili linklemeyi iyileştir (ilgili yazılar, kategoriler, breadcrumb).
- [ ] Sayfalama SEO'sunu iyileştir (temiz URL'ler ve indexleme stratejisi).
- [ ] Görsel SEO'yu iyileştir (açıklayıcı alt metin, boyut optimizasyonu, lazy loading kontrolleri).
- [ ] Blog sayfalarında Core Web Vitals metriklerini iyileştir (LCP, CLS, INP).

## P3 - İzleme ve Yönetişim

- [ ] Site geneli yapılandırılmış veri ekle (Organization ve WebSite).
- [ ] Google Search Console kontrollerini kur (kapsama, sitemap, tarama sorunları).
- [ ] İnce içerik politikasını tanımla (düşük değerli sayfaları birleştirme/noindex).
